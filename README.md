# vscode-themer
A Figma plugin to design VS Code themes

## Demo
<img src="./assets/demo.gif">

## Build
Run `tsc -w` or if you're using [VS Code](https://code.visualstudio.com/) you can use `Cmd/Ctrl+Shift+B` to trigger the default build task.
